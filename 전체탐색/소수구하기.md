## 소수 구하기
> 문제
```
M이상 N이하의 소수를 모두 출력하는 프로그램을 작성하시오.
```
> 시도
```cpp
#include<iostream>
#include<vector>
#include<list>
using namespace std;

class Decimal{
public:
	list <int> find_decimal(int a, int b){
		list<int> list1;

		for(int i=a; i<=b; i++)
			list1.push_back(i);
		
		for(int i=2; i<b; i++){
			for(list<int>::iterator it = list1.begin(); it != list1.end(); ++it){
				
				if( *it != i && *it%i == 0 ){
					list1.erase(it--);
				}
			}
		}
		return list1;
	}
};

int main(){
	Decimal deci;
	list<int> list1;
	int a, b;
	cin >> a;
	cin >> b;
	list1 = deci.find_decimal(a, b);
	
	for(list<int>::iterator it = list1.begin(); it != list1.end(); ++it){
		cout << *it << endl;
	}
}
```
> 문제점
```
위와 같이 cpp의 list를 연결리스트라고 생각하고 에라토스테네스의 체의 원리로
i의 배수가 되는 ( 소수가 아닌 )수는 리스트에서 삭제하는 방식을 사용했다.

그런데 시간 초과가 되었다.
아마 이중포문에서 사용한 it != list1.end() 의 list1.end()를 매번 계산해서
시간 초과가 발생하지 않았나 싶다.
list1.end()를 저장하여 사용하는 등의 수정을 해보았지만 컴파일 에러가 발생하여 포기했다. 
```
